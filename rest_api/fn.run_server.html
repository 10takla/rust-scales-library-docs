§Операции с продуктами

  • POST /products/set

    • path: Путь к JSON файлу (String)
    • remove_product_id: Флаг для выполнения autoincrement запроса (bool)
    • clear_all: Флаг для удаления всех записей по id, указанным в JSON (bool)

  • GET /products/{product_id}/draw-label

    • path: Опциональный путь к файлу для сохранения этикетки (Option). По умолчанию сохраняется в <server>/output
    • label_template_id: Опциональный идентификатор шаблона этикетки (Option). При отсутствии используется product.label_template

§Тестовые функции

  • GET /test/calc-sin

  • GET /test/calc-sin-thread

    • count: Количество вычислений синуса в многопоточном режиме (i32)
